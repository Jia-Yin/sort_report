一、操作原理

選擇排序法（Selection Sort）是一種簡單直觀的排序演算法。
核心概念：每一輪從未排序區間中選出最小值，放到最前面， 依序將最小值「選出」並放到正確位置。

每完成一輪，未排序區間會縮小一格，直到全部排序完成。

二、演算法偽代碼

function selectionSort(arr):
  1n = arr.length
  2for i = 0 to n-2: // 共 n-1 輪
  3  minIdx = i
  4  for j = i+1 to n-1:
  5    if arr[j] < arr[minIdx]:
  6      minIdx = j
  7  if minIdx != i:
  8    swap(arr[i], arr[minIdx])

三、時間 & 空間複雜度

情況 時間複雜度 說明
最佳情況 (Best) O(n²) 無論初始狀態，皆需 n-1 輪
平均情況 (Average) O(n²) 元素隨機分布
最壞情況 (Worst) O(n²) 陣列完全逆序
空間複雜度 O(1) 原地排序，僅需常數額外空間
穩定性 不穩定 ✗ 相等元素可能改變原始順序
